Glamorous Toolkit is the moldable integrated development environment. For Pharo. It is made of components that are programmable live while using the tool itself. And it is open-source under an MIT license.

Install and run
on Mac, Windows or Linux

What's interesting about Glamorous Toolkit? It offers a fundamentally new perspective on programming. We call it moldable development. We do have cool widgets and fancy code analyses, but our goal is to change the way you experience software. You can read about it, but to understand it, you have to play with it. And engage with us. We'd love to hear from you.
